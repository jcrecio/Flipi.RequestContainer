1. The image for the python flask api needs to be built and pushed into the registry
docker login

docker build -t jcrecio/flipi-requestcontainer-swarm_webapi:latest -f ./Dockerfile .
docker push jcrecio/flipi-requestcontainer-swarm_webapi:latest


2. Start the swarm (if not yet)
docker swarm init

3. Deploy into the swarm:
docker stack deploy --compose-file docker-compose.yml flipi
